Timeline of Computing

1940s – Early Machines

Early machines like ENIAC used pure machine code (1s and 0s). No compilers, no high-level languages.

1950s – Assembly Language Emerges

Assembly allowed symbolic names instead of binary. A massive breakthrough in usability.

1970s – Microprocessors

Assembly became essential for early consumer computers and embedded systems.

Today – Still Relevant

Used in cybersecurity, reverse engineering, embedded devices, and performance-critical software.

What Is Assembly?

Assembly is a low-level language that interacts directly with CPU registers, memory, and hardware operations.

Example (x86 Assembly):

MOV AX, 5
MOV BX, 3
ADD AX, BX
; AX now contains 8

This gives direct control over hardware — something high-level languages hide.
